Put project JPG files in this folder using this naming format:

<project-id>-1.jpg
<project-id>-2.jpg
<project-id>-3.jpg
<project-id>-4.jpg

Examples:
spatial-console-1.jpg
jarvis-1.jpg
stark-paper-analyzer-1.jpg
code-vault-1.jpg

Project IDs currently used in src/components/Projects.jsx:
spatial-console
jarvis
stark-paper-analyzer
code-vault
air-canvas
pose-detection
anime-clash
bomb-difuse

Notes:
- Use lowercase IDs exactly as above.
- Keep file extension as .jpg.
- The UI cycles 1 to 4 with wrap-around.
